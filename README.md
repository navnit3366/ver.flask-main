### 연구과제
- 각 채용공고를 눌렀을 때 더 자세히 볼 수 있도록 만들기
- django프레임워크를 사용하여 구현하기
- 우리학교 학생들만 조회가능하도록 하기
- 실제 웹서비스로 올리기
